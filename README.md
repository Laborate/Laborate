Clone To Existing Directory
-----------------------------
```bash
git clone git@github.com:bvallelunga/Codelaborate.git tmp; mv tmp/* tmp/.git* .; rm -R tmp/
```

```bash
shell/local/start.sh
```

**stop node**
```bash
shell/local/start.sh
```

```bash
shell/server/start.sh
```
